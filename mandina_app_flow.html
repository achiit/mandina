Zone 01

Pre-Auth Experience

01
MVP — Phase 1
SCREEN 01
Splash / Logo Screen
~2.5 sec

First touchpoint. Sets the premium tone of the app. Plays on every cold launch.

  • Dark navy full-screen background
  • Mandina logomark appears with a slow scale-up + fade-in (800ms ease)
  • Wordmark fades in below logo with letter-spacing animation
  • Thin gold line draws underneath the wordmark (stroke animation)
  • Gentle radial glow pulses once, then fades
  • Auto-advances to Onboarding (first time) or Dashboard (returning users)
→ First-time users go to Onboarding · Returning users skip to Dashboard
SCREEN 02
Onboarding — Slide 1
Swipeable

Introduces the core value proposition. Slide 1 of 3 in an auto-play carousel.

  • Illustration: A cosy home with a glowing chat bubble emerging from it
  • Headline: "Your Home Has an Expert on Call"
  • Body: Short 1-line benefit statement
  • Swipe dots indicator at bottom (3 dots)
  • Skip button (top-right) takes to Auth Gateway
  • Slide transitions with a gentle horizontal parallax
SCREEN 03
Onboarding — Slide 2
Swipeable

Explains service coordination — the second core pillar.

  • Illustration: Calendar, wrench and a handshake icon in motion
  • Headline: "We Find, Book, and Manage It All"
  • Body: One-line emphasis on zero hassle
  • Animated icon loop (subtle bounce every 3 seconds)
  • Same dot + skip navigation
SCREEN 04
Onboarding — Slide 3
Last Slide

Introduces the Home Passport concept. Ends with a clear CTA.

  • Illustration: A digital "passport book" opening to reveal appliance icons
  • Headline: "Your Home's Complete Digital Record"
  • Body: One-line about tracking history and future repairs
  • Primary CTA button: "Get Started" — navy pill button
  • Secondary link: "I already have an account"
  • CTA button scales up with a subtle spring animation on tap
→ "Get Started" → Sign Up · "Have account" → Login
Zone 02

Authentication

02
MVP — Phase 1
SCREEN 05
Auth Gateway
Entry Point

Clean dual-action screen. No form fields yet — just intent selection.

  • Mandina logo (compact) centred at top
  • Large button: "Create Account" (filled, navy)
  • Large button: "Sign In" (outlined, navy border)
  • Divider or "Continue with" section (Google / Apple Sign-In — Phase 2)
  • Light background, minimal, premium feel
SCREEN 06
Sign Up Form
New Users

Minimal sign-up form. Collects only what's needed to get started.

  • Fields: Full Name, Email Address, Mobile Number (with country code picker)
  • Password field with show/hide toggle
  • Terms & Privacy checkbox (required)
  • CTA: "Send Verification Code"
  • Keyboard-aware scrolling so form never hides behind keyboard
  • Real-time field validation with soft red/green indicators
→ On submit → OTP Verification Screen
SCREEN 07
OTP Verification
Security

6-digit OTP sent to mobile. Simple, focused screen.

  • Masked phone number shown ("Sent to +44 ••• •••• 1234")
  • 6 individual input boxes that auto-advance on digit entry
  • Auto-read SMS OTP (with OS permission prompt)
  • Countdown timer: "Resend in 0:59"
  • Resend link activates after timer expires
  • Wrong OTP shows subtle shake animation on input row
→ Valid OTP → Home Setup (new) or Dashboard (returning)
SCREEN 08
Sign In Screen
Returning Users

Fast login for returning users. Biometric login offered after first sign-in.

  • Email + Password fields
  • "Forgot Password?" link (triggers email reset flow)
  • Face ID / Touch ID option if previously enabled
  • Biometric login: one-tap icon → instant unlock
  • Sign In button
  • "Don't have an account?" link to Sign Up
→ Authenticated → Dashboard
Zone 03

Home Setup — Home Passport Init

03
MVP — Phase 1
SCREEN 09
Welcome + Name
First Time Only

Personal welcome moment before entering data. Builds warmth.

  • "Welcome, [First Name]" — large serif heading, staggered fade-in
  • Short copy: "Let's set up your Home Passport so we're ready when you need us."
  • Animated illustration of a home being "unlocked"
  • CTA: "Let's Begin" → progresses to Property Setup
  • This screen is shown ONCE only — never again after initial setup
SCREEN 10
Property Setup
Home Passport

Captures the home's basic details. This becomes the foundation of the Home Passport.

  • Full address input (with address autocomplete from Google Places API)
  • Property type selector: House / Apartment / Condo / Townhouse (icon tiles)
  • Year built (optional — picker wheel)
  • Approx. square footage (optional — slider)
  • Number of bedrooms / bathrooms (stepper input)
  • Ownership type: Owner / Renter (affects feature access)
  • Progress bar: Step 1 of 3
SCREEN 11
Appliance & Systems Setup
Home Passport

User selects categories of appliances/systems they have. Details can be filled in gradually.

  • Grid of category tiles: HVAC, Plumbing, Electrical, Kitchen, Laundry, Security, Garden
  • Tap any tile to expand it and add item details (Brand, Model, Year of install, Warranty expiry)
  • "Skip for now" link — user can come back later via Home Passport screen
  • Each filled-in category shows a green checkmark on tile
  • Progress bar: Step 2 of 3
SCREEN 12
Setup Complete
Completion

Celebratory moment. Reinforces value before dropping the user into the app.

  • Animated checkmark or passport-stamp animation (Lottie)
  • Headline: "Your Home Passport is Ready"
  • 3 benefit pills shown: "Instant Expert Access · Managed Bookings · Smart Maintenance"
  • CTA: "Take Me Home" → navigates to Dashboard with a full-screen slide transition
→ → Main Dashboard (Home Screen)
Zone 04

Main Dashboard — Home Screen

04
MVP — Phase 1
SCREEN 13
Home Dashboard
Core Screen

The central hub of the app. Gives the user a snapshot of their home and quick access to everything.

  • Header: "Good morning, [Name]" + address of active property + avatar
  • Primary CTA Card: Large navy card — "Get Help Now" with concierge icon and availability indicator ("3 experts online")
  • Active Issues Strip: Horizontally scrollable cards for any open cases
  • Upcoming Services: Next scheduled service date with provider name
  • Maintenance Alerts: Yellow alert cards (e.g., "Boiler service due in 14 days")
  • Home Passport shortcut: Mini passport card showing last updated date
  • Bottom Tab Bar: Home · Issues · Passport · Bookings · Profile
SCREEN 13b
Notifications Centre
Bell Icon

Accessible from the bell icon on the Dashboard header.

  • Chronological notification list with category icons (Maintenance, Booking, Payment, Concierge)
  • Unread notifications have a subtle gold left border
  • Tap notification → navigates to the relevant screen
  • Swipe left on any notification to dismiss
  • "Clear All" option at top right
Zone 05

Issue Submission Flow

05
MVP — Phase 1
SCREEN 14
Issue Category Selection
Step 1 of 3

User describes what's wrong. Guided category selection removes guesswork.

  • Header: "What's the problem?" — calm, reassuring tone
  • 6 category tiles with icon + label: Plumbing · Electrical · HVAC · Appliance · Structural · Other
  • Selected tile gets a gold border highlight + check icon
  • Sub-category appears below on selection (e.g., under Plumbing: Leak, Blockage, No Hot Water, etc.)
  • Urgency indicator: "Is this an emergency?" toggle with red highlight
  • Next button activates once category + sub-category are selected
SCREEN 15
Describe + Media Upload
Step 2 of 3

User adds more detail and can attach photos or video of the issue.

  • Text area: "Describe what's happening…" (with character count)
  • Media strip: Tap to add photos or a short video (max 30 sec)
  • Thumbnail preview of attached media with remove option
  • Which property / room selector (pre-populated from Passport)
  • Optional: "Which appliance is affected?" (pulls from Home Passport)
  • Submit button: "Find Me an Expert"
SCREEN 16
Matching / Finding Expert
Step 3 of 3

A brief animated loading screen while the system routes the request to an available concierge.

  • Animated pulsing concierge avatar (Lottie loop)
  • Status message cycles: "Reviewing your issue…" → "Finding the right expert…" → "Connecting you now…"
  • Target wait time indicator: "Typical wait: under 5 minutes"
  • Option to receive a push notification when expert is ready (instead of waiting on screen)
  • Cancel option at bottom
  • Auto-advances when concierge accepts the case
→ Expert accepted → Live Chat Screen
Zone 06

Live Concierge Interaction

06
MVP — Phase 1
SCREEN 17
Live Chat Interface
Core Feature

The primary interaction screen. Real-time chat between user and the concierge expert.

  • Expert header: photo, name, role title ("Home Expert"), online indicator
  • Chat bubbles: User (right, navy) · Expert (left, cream/white)
  • Typing indicator (animated dots) when expert is responding
  • Message types: Text · Photo · Quick-reply options (expert can send structured answers)
  • Paperclip icon to attach additional photos mid-chat
  • "Switch to Video" button in header (upgrades interaction)
  • Issue summary pinned at top (expandable)
  • Expert diagnosis card can be sent inline (structured card in chat)
SCREEN 18
Video Consultation
Upgrade Option

User shows the issue live to the concierge via camera. Reduces misdiagnosis significantly.

  • Full-screen video call UI (WebRTC-based)
  • Expert's face in small pip (corner), user's camera fills screen
  • User can flip camera to show the issue area
  • Torch/flashlight toggle (for dark areas like under-sink)
  • Mute and end-call buttons
  • Expert can capture a screenshot of the issue with user permission
  • On call end → returns to chat with a call summary card
SCREEN 19
Expert Diagnosis Card
Outcome

Expert sends a structured diagnosis card inside the chat. This is the key deliverable from the concierge interaction.

  • Issue Summary: What was reported
  • Expert Assessment: What the expert believes the problem is
  • Urgency Level: 🔴 Urgent · 🟡 Soon · 🟢 When convenient
  • Recommended Trade: Plumber / Electrician / HVAC Tech, etc.
  • Rough Price Range: "Estimated £80–£180 for this type of repair"
  • Two action buttons: "Find Me a Service Provider" · "I'll Handle It Myself"
  • Card is saved permanently to case history
→ "Find Me a Provider" → Service Recommendation Screen
Zone 07

Service Recommendation & Booking

07
MVP — Phase 1
SCREEN 20
Service Provider Recommendations
Curated List

Concierge-curated list of vetted providers for this specific issue type. Not a generic marketplace.

  • Header: "Recommended for [Issue Type]" + expert's avatar (showing it's personally vetted)
  • Provider cards: Photo, Name, Trade, Star Rating, Number of jobs, Estimated availability
  • Price estimate per provider (if available): shown as a range
  • Verified badge on all providers
  • "Why this provider?" expandable section with concierge note
  • Tap card → Provider Detail Screen
  • 3–5 providers shown maximum (not overwhelming)
SCREEN 21
Provider Detail
Profile

Deep profile of a specific provider before the user commits to booking.

  • Photo, name, trade category, years of experience
  • Star rating with review count + recent review snippets
  • Verified certifications / licences
  • Service area shown on mini-map
  • Earliest available slot highlighted
  • Price guide (call-out fee + hourly / fixed)
  • CTA: "Book This Provider"
SCREEN 22
Date & Time Selection
Booking Step 1

Calendar-based slot picker. Shows real availability pulled from provider.

  • Monthly calendar view with available days highlighted in gold
  • On date selection, time slots appear below (morning / afternoon / evening)
  • Slot pills: available (tappable) vs booked (greyed out)
  • Time slots show duration estimate (e.g., "60–90 min job")
  • Confirm button at bottom: "Book for [Day, Time]"
SCREEN 23
Booking Confirmation
Booking Step 2

Final review before payment. User sees everything clearly before committing.

  • Summary card: Provider, Date/Time, Issue type, Address
  • Estimated cost range (where available)
  • Special instructions text field (optional)
  • Access instructions: "How should the provider enter?" (Key safe, ring bell, etc.)
  • Payment method selector (if payment is required upfront)
  • CTA: "Confirm Booking"
  • Booking is managed by the concierge team — highlighted reassurance line
→ Confirmed → Payment Screen (if applicable)
Zone 08

Payments & Trust Layer

08
MVP — Phase 1 (Basic) + Phase 2 (Escrow)
SCREEN 24
Payment Screen
Secure

Handles subscription payment (base plan) and/or service call-out fee payment.

  • Amount shown prominently with breakdown (service fee, any platform fee)
  • Saved card display (last 4 digits + card type icon)
  • Add new card (Stripe card input form)
  • Apple Pay / Google Pay one-tap option
  • Payment protection option: "Enable Escrow — pay released after job completion" (Phase 2)
  • Secure payment badge (SSL / Stripe powered label)
  • CTA: "Pay Securely"
SCREEN 24b
Payment Success
Confirmation

Reassuring post-payment confirmation. Sets expectations for what comes next.

  • Animated green checkmark / Lottie success animation
  • Booking reference number
  • "What happens next" steps (3 bullet timeline): Concierge confirms · Provider notified · You get a reminder
  • Invoice download link (PDF)
  • CTA: "Go to My Bookings" or "Back to Home"
→ → Bookings tab / Dashboard
Zone 09

Day-of Service & Live Tracking

09
MVP — Phase 1 (Status) · Phase 2 (Live Map)
SCREEN 25
Pre-Visit Card
Sent Day Before

Push notification sent the evening before the visit. Tapping it opens this screen.

  • Provider's name and photo (knowing who's coming to your door)
  • Scheduled time window (e.g., "Between 10am – 12pm")
  • Provider's vehicle description (optional, Phase 2)
  • Contact button: Tap to message or call provider
  • Reschedule option (if needed, with 24hr notice)
  • Safety note: "All Mandina providers are vetted and verified"
SCREEN 26
Live Tracking / Status Screen
Day of Visit

Tracks the status of the service visit from provider on-the-way to job complete.

  • Status timeline: Confirmed → Provider En Route → Arrived → In Progress → Complete
  • Current step highlighted with animated pulse
  • ETA shown when en route: "Arriving in ~18 minutes" (Phase 2: live map view)
  • Push notification sent at each status change
  • Contact provider button throughout
  • On "Complete" status: Rate & Review prompt appears
SCREEN 27
Post-Service Review & Feedback
Completion

Captures feedback immediately after job completion. Feeds back into platform quality.

  • Star rating (1–5) for: Provider Quality, Punctuality, Value for Money
  • Optional written review (free text)
  • Rating for concierge advice accuracy: "Was the expert's assessment correct?"
  • Photo upload: option to document the finished repair
  • "Would you use this provider again?" Yes / No
  • Submit button → triggers invoice delivery and case closure
→ Case closed → Logged in History & Home Passport
Zone 10

History, Home Passport & Profile

10
MVP — Phase 1
SCREEN 28a
Issues History
Issues Tab

Full chronological log of all past and open cases. The user's home service diary.

  • Filter tabs: All · Open · Resolved
  • Each case card: issue type icon, title, date, status badge, provider name
  • Tap case → Case Detail Screen (full chat log, diagnosis, invoice, photos)
  • Search bar (search by issue type, provider, date)
  • Export all history as PDF option (Profile settings)
SCREEN 28b
Home Passport
Passport Tab

The digital record of the user's home. Living document that updates after every service.

  • Property summary header: address, type, year built
  • Appliances & Systems list (expandable rows): brand, model, install year, warranty status, last serviced
  • Add / Edit any appliance at any time
  • Maintenance forecast: "Boiler service recommended by March 2026" (rule-based initially)
  • Annual Home Condition Review card (Phase 2 — human assessment)
  • Health / Accessibility notes section (higher subscription tier)
SCREEN 28c
Profile & Settings
Profile Tab

Account management, household members, subscription, and preferences.

  • User avatar + name + subscription tier badge
  • Manage Household Members: Add/remove users, assign permissions (view-only / full access)
  • Subscription Plan: Current plan, upgrade options
  • Saved payment methods
  • Notification preferences
  • Privacy & Data settings
  • Help & Support link → opens concierge chat (non-issue support)
  • Sign Out
